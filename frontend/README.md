# Repertorio Musical Frontend

Next.js frontend for the Repertorio Musical app.

## Features

- Public song search
- Internal chord pages
- Login/register
- User repertories
- Add/remove songs from repertories
- Chord transposition
- Saved key per repertory song

## Scripts

```bash
npm run dev
npm run build
npm run start
npm run lint
```

The development server runs on port `3000` by default.

Set the backend URL with:

```bash
NEXT_PUBLIC_BACKEND_URL=http://localhost:3333
```
